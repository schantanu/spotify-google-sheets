Playlists Summary

Get an overview of all your playlists including:

  • Total tracks
  • Playlist owners
  • Quick access links
  • Privacy settings

Import Tracks

Download tracks data from:

  • Liked Songs library
  • Your personal playlists
  • Playlists you follow
  • Public playlists

Update Playlists

Modify and manage your playlists from Google sheets:

  • Add tracks using Track IDs
  • Remove tracks by deleting Track IDs
  • Reorder tracks by moving Track IDs
  • Bulk edit multiple playlists

Create Playlists

Create new playlists easily:

  • Name your playlist
  • Add an optional description
  • Add tracks from sheets

Note: Due to Spotify API limitations, algorithmic playlists and Spotify-owned editorial playlists are not supported.
